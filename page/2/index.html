Kmeans

发表于 2022-01-15 分类于 深度学习

Kmeans是一个原理较为简单的聚类模型，它的操作步骤是

  1. 随机选择k个点作为初始类心。
  2. 计算每个元素和k个类心之间的距离并归类到最近的类里面。
  3. 以每个类的均值作为新的类心。
  4. 重复2和3知道所有的类心不再变化。
